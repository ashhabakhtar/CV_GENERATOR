**Smart CV Maker** is a professional-grade resume builder designed to bridge the gap between stunning visual aesthetics and rigid Applicant Tracking System (ATS) compatibility. Built with React and Tailwind CSS, it empowers job seekers to create resumes that impress both humans and machines.

---

## ✨ Key Features

### 🔍 ATS Optimization Engine
*   **Real-time Keyword Matching**: Paste a job description and instantly see your resume's compatibility score.
*   **Skill Gap Analysis**: Identifies missing critical keywords and suggests optimizations.
*   **Structural Audit**: Automatically checks for standard section headers, contact info presence, and content density.

### 🎨 Curated Template Gallery
Switch between professional layouts with a single click (respecting the "ATS Safe Mode" toggle):
*   **Modern Premium**: Glassmorphism UI for tech and design roles.
*   **Classic Professional**: Centered, serif-based layout for formal industries (Law, Finance).
*   **Creative Minimalist**: Bold asymmetric designs with integrated skill progress bars.
*   **ATS Minimalist**: Ultra-safe, single-column design optimized for maximum parsing success.

### 📄 Premium PDF Generation
*   **Selectable Text**: High-quality PDF export using `@react-pdf/renderer`.
*   **Machine-Readable**: Ensures all content is correctly parsed by LinkedIn, Workday, and Greenhouse.
*   **Live Preview**: See your changes update instantly in the side-by-side editor.

## 🛠️ Tech Stack

- **Frontend**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: Custom React Hooks + Local Storage
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PDF Engine**: [@react-pdf/renderer](https://react-pdf.org/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ashhabakhtar/CV_GENERATOR.git
   cd CV_GENERATOR
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
