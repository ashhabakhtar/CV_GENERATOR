**Smart CV Maker** is a professional-grade, mobile-optimized resume builder designed to bridge the gap between stunning visual aesthetics and rigid Applicant Tracking System (ATS) compatibility. Built with React, TypeScript, and Tailwind CSS, it empowers job seekers to create resumes that impress both human recruiters and automated machines.

---

## 🔥 New in Elite Edition

### 🤖 Smart AI-Powered Parser (Local)
No more manual typing! Upload your old PDF resume and let our local heuristic engine extract your Personal Info, Work History, Education, and Skills instantly. **100% Client-Side Privacy** – your data never leaves your browser.

### 📱 Mobile-First Excellence
The entire platform has been re-engineered for mobile devices. Use the professional bottom-navigation bar and touch-friendly interface to build your resume on the go. Optimized typography prevents browser "auto-zoom" for a native-app feel.

### 📊 1-5 Skill Proficiency System
Visualise your expertise with our new 1-5 star rating system, rendered beautifully across all premium templates.

---

## ✨ Core Features

### 🔍 ATS Optimization Engine
*   **Real-time Keyword Matching**: Paste a job description and instantly see your resume's compatibility score.
*   **Missing Keyword Analysis**: Identifies critical technical and soft skills you should include.
*   **Structural Auditing**: Automatically warns about date inconsistencies or missing contact fields.

### 🎨 Premium Template Gallery
Switch between 6+ professional layouts with a single click:
*   **Modern Premium**: Glassmorphism UI for tech and design roles.
*   **Professional Modern [NEW]**: Sleek, high-contrast layout with sidebar.
*   **Clean Minimalist [NEW]**: Ultra-clean professional design for minimalists.
*   **ATS Minimalist**: Ultra-safe design optimized for 100% parsing success across all ATS portals.

### 📄 Elite PDF Generation
*   **Machine-Readable Text**: High-quality PDF export using `@react-pdf/renderer`.
*   **Live Global View**: See your resume update instantly in the side-by-side preview pane.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS (Mobile-Optimized)
- **Animations**: Framer Motion
- **Parsing**: PDF.js (Client-side heuristics)
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ashhabakhtar/CV_GENERATOR.git
   cd CV_GENERATOR
   ```

2. **Install & Run:**
   ```bash
   npm install
   npm run dev
   ```

3. **Visit Your Creation:**
   Open [http://localhost:5173/CV_GENERATOR/](http://localhost:5173/CV_GENERATOR/)
